\# PRIORIZA – Casos de Uso



\## Actor principal

\- Usuario



---



\## CU-01 Crear usuario

\*\*Descripción:\*\*  

El usuario se registra en la aplicación indicando su correo electrónico.



\*\*Flujo principal:\*\*

1\. El usuario abre la aplicación.

2\. Introduce su email.

3\. El sistema valida el formato.

4\. El sistema crea el usuario.



---



\## CU-02 Crear tarea

\*\*Descripción:\*\*  

El usuario crea una nueva tarea.



\*\*Flujo principal:\*\*

1\. El usuario selecciona “Nueva tarea”.

2\. Introduce título, descripción y fecha límite.

3\. El sistema guarda la tarea.

4\. El motor heurístico asigna prioridad.



---



\## CU-03 Editar tarea

\*\*Descripción:\*\*  

El usuario modifica una tarea existente.



---



\## CU-04 Eliminar tarea

\*\*Descripción:\*\*  

El usuario elimina una tarea.



---



\## CU-05 Clasificar tareas automáticamente

\*\*Descripción:\*\*  

El sistema clasifica tareas como urgente, importante o lejana.



---



\## CU-06 Generar informes

\*\*Descripción:\*\*  

El usuario genera informes de tareas.



---



\## CU-07 Enviar alertas por email

\*\*Descripción:\*\*  

El sistema envía alertas cuando una tarea es urgente o próxima a vencer.
